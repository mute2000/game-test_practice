# Bingo Frenzy 游戏深度理解主文档

## 1. 游戏基础信息
### 1.1 游戏基础信息
- 游戏类型：休闲竞技类 Bingo游戏
- 目标用户：休闲手游玩家，喜欢轻竞技、碎片化娱乐的人群

### 1.2 核心玩法内核
以经典宾果数字连线为基础，融合：
- 多卡牌同步对局（1/2/4/6 张）
- 图案触发 Buff（carefulness / studiousness / efficiency）
- 特殊机制（super power-up）
- 多种玩法（frozen catch 等）
- 活动体系（crabby ko!、SOS、welcome home、团队协作活动等）
- 社交模块（Club、对战、排行榜）

---

## 2. 核心玩法机制
### 2.1 基础对局规则
1. 单局可选择1/2/4/6张宾果卡同步进行，可选择不同倍数，卡牌越多，连线概率越高，但操作压力越大
2. 系统随机播报数字，玩家手动标记，完成横/竖/斜直线或四角触发Bingo
3. 多次错误标记点未播报数字会触发 -40 惩罚，会影响排名
4. 对局结束，结算奖励并更新美食碎片解锁和活动进度

### 2.2 游戏资源说明
- ticket：对局入场资源，可通过 Daily Bonus、充值、活动、签到获取
- diamonds（钻石）：可通过 Daily Bonus、充值、活动、签到获取；用于购买power-up，活动gold rush机会
- power-up（道具）：增加bingo概率

### 2.3 成长与进度系统
- 等级提升规则：通过对局，可获得经验，经验条满后即可升级
- 等级解锁规则：等级提升后，下注倍数上限提高，可使用的宾果卡数量增加，可参与的活动增多，解锁新的特色玩法
- 美食解锁规则：对局获胜后可获得碎片，碎片集齐后，可解锁新美食

## 4. 游戏体验总结
### 4.1 优点
- 美食旅行主题与 Bingo 结合度高，解锁不同城市美食的成就感强，弱化了 Bingo 的“纯随机”枯燥感
- 活动和玩法多样，提升游戏趣味性，降低对游戏的疲劳感
- 操作简单，适配碎片化时间，可随时随地进行对局
- power-up和pattern玩法丰富，合理使用道具提升 Bingo 概率，具备轻度策略性
- super power-up是限时的，可有效提升玩家留存

### 4.2 体验不足
- 游戏整体布局杂乱，活动上下左右都有
- 有引导消费的情况
- 部分活动引导文字繁琐，不够简单易懂
- back to map后弹窗过多，影响游戏体验
- 邮件系统无“一键领取”，操作繁琐
